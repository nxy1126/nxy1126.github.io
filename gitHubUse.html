GitHub使用方法

GitHub是一个面向开源及私有软件项目的托管平台,也是一个仓库

  1. 首先,打开终端,在桌面上使用mkdir + "文件名"创建一个空的文件夹用来存放github;
  2. 然后用cd方法进入自己创建的文件夹,使用git init来初始化github仓库和创建一个主线;
  3. 建立用户名:git config --global user.name "用户名";
  4. 建立邮箱名:git config --global user.email "邮箱名";
  5. 打开文件夹或者在终端输入touch + "文件名"
  6. 使用git add方法把文件加入上传仓库的列表,git add后面可以加具体的文件名,表示只上传这个文件;也可以加".",表示当前文件夹内所有文件都上传
  7. 在使用git add之后可以用git status来查看当前上传列表的文件
  8. git commit -m "注释"方法确认提交
  9. 可以使用git log来查看确定提交的文件
  10. 然后把自己在github.com创建的New repository中的关联代码粘贴到终端;
  11. 回车确定,就可以在github访问或下载自己上传的文件;
  • 分支
  • 当需要分支时,在终端输入git branch查看当前主线是否有分支
  • 如果没有,可以输入git branch + "分支名"来创建分支
  • 输入命令git checkout + "分支名"进入分支,可以和主线一样创建,上传文件
  • 如果需要回到主线,输入git checkout master
  • 主线和支线合并输入需要在主线状态下输入git merge + "分支名"
